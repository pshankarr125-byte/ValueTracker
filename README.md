# ValueTracker

## Overview
ValueTracker is an AI-powered personal finance web application that helps users track their budget and receive personalized spending advice.

## Features
- Budget and expense tracking
- AI-powered financial advice
- Savings goal tracking
- Future spending insights

## Tech Stack
- Python
- Flask
- HTML
- CSS
- Google Gemini API

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/pshankarr125-byte/ValueTracker.git
```

### 2. Open the project folder

```bash
cd ValueTracker
```

### 3. Install the required packages

```bash
pip install -r requirements.txt
```

### 4. Add your Gemini API key

Create a `.env` file or set an environment variable:

```text
GEMINI_API_KEY=YOUR_API_KEY
```

### 5. Run the application

```bash
python app.py
```

### 6. Open your browser

```
http://127.0.0.1:5000
```
